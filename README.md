# goal-project-pyramid
# 目標企画ピラミッド

目標から企画、具体的な行動までを4階層で整理するためのブラウザツールです。

配信企画、動画制作、ブログ記事、個人プロジェクトなどの設計に利用できます。

## 主な機能

* ビジョン・ミッションの整理
* 中長期の方向性の整理
* 現在のテーマと行動目標の設定
* 企画カードの作成
* Markdown形式での保存
* Markdown形式での読み込み
* JSON形式での保存と読み込み
* 入力内容のブラウザ保存
* 印刷対応

## 使い方

GitHub Pagesの公開ページをブラウザで開き、各項目を入力してください。

入力した内容は使用中のブラウザ内に保存されます。

## ローカルで使う方法

1. Githubの画面上部の「Code」を押す
2. 「Download ZIP」を選ぶ
3. ダウンロードしたZIPファイルを展開する
4. 展開したフォルダー内の `index.html` を開く

インストールは不要です。

## データについて

入力内容は外部サーバーへ送信されません。

ブラウザのローカルストレージに保存されるため、ブラウザのデータを削除すると入力内容も消える場合があります。必要な内容はMarkdownまたはJSONで保存してください。
